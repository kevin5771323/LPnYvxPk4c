# 宠物管理系统 java165

## 项目概述

宠物管理系统是一款基于B/S架构、MVC模式的前后端分离系统。该系统致力于为用户提供宠物领养、救助信息发布与管理等功能，同时为管理员提供全面的宠物信息管理能力。

## 技术栈

- **前端**: Vue
- **后端**: Java、SpringBoot、Mybatis-Plus
- **数据库**: Mysql
- **开发工具**: IDEA、JDK1.8、Maven

## 功能模块

### 用户端功能

- **首页**
  - 展示公告列表
  - 宠物科普
  - 介绍流浪宠物
  - 热门活动展示
- **宠物领养**
  - 搜索并申请领养宠物
  - 查看已领养宠物信息
- **宠物救助**
  - 展示需要救助的流浪宠物
  - 新增流浪宠物信息
  - 展示宠物救助站信息
  - 查看需要喂养的流浪宠物地点和环境
- **丢失宠物查看**
  - 查看丢失宠物及失主信息
- **宠物论坛**
  - 交流宠物领养心得
  - 发表及评论帖子

### 管理员端功能

- **流浪动物管理**
  - 管理流浪宠物信息
  - 发布流浪宠物信息
- **待绝育动物管理**
  - 管理流浪动物健康信息
- **评论管理**
  - 管理宠物论坛用户评论
- **流浪动物救助**
  - 发布救助信息
  - 处理已解决救助
- **喂养点管理**
  - 管理宠物喂养点
- **动物走失管理**
  - 管理丢失的动物信息
- **救助站管理**
  - 管理流浪动物救助站
- **帖子管理**
  - 管理宠物论坛帖子
